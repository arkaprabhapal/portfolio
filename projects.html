Projects by Arkaprabha Pal

This projects are done by Arkaprabha Pal, 1st Year Electronics and Telecommunication Engineering Ungergraduate Student at Jadavpur University.

Line Follower Robot

This robot automatically follows a Black Line on a white surface. It senses the colours through IR sensor. Other major components are Arduino UNO R3, L293C Motor Driver Shield etc.

Obstacle Avoidance Robot

This robot moves with avoiding all possible collisions which could occur with it. It uses Ultrasonic distance sensor, IR Module, Arduino UNO R3.

Human Following Robot

This Robot follows human being and it uses Ultrasonic distance sensor, IR Module, Arduino UNO R3 etc.

Student Automatic Attendance System

This is a Basic project of IoT. It uses RFID Card, NodeMCU 1.0 to automatically upload student name, time, date in the database when student's card is scanned. I have used google sheets for this project.
